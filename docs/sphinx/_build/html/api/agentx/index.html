agentx

AgentX - Multi-Agent Conversation Framework

A flexible framework for building AI agent teams with: - Autonomous agents with private LLM interactions - Centralized tool execution for security and monitoring - Built-in storage, memory, and search capabilities - Team coordination and task management

class agentx.Agent(config: agentx.core.config.AgentConfig, tool_manager=None)[source]

Represents an autonomous agent that manages its own conversation flow.

Key Principles: - Each agent is autonomous and manages its own conversation flow - Agents communicate with other agents through public interfaces only - The brain is private to the agent - no external access - Tool execution is handled by orchestrator for security and control

This combines: - AgentConfig (configuration data) - Brain (private LLM interaction) - Conversation management (delegates tool execution to orchestrator)

Initialize agent with configuration and optional tool manager.

Parameters:

  • config – Agent configuration

  • tool_manager – Optional tool manager (injected by TaskExecutor)

agentx.start_task(prompt: str, config_path: str, task_id: str | None = None, workspace_dir: pathlib.Path | None = None) → TaskExecutor[source]

High-level function to start a task and return the TaskExecutor for step-by-step execution.

This function is ideal for interactive scenarios where you want to: - Execute tasks step by step - Inspect task state between steps - Modify task configuration during execution - Build interactive UIs with manual control

Parameters:

  • prompt – The initial task prompt

  • config_path – Path to the team configuration file

  • task_id – Optional custom task ID

  • workspace_dir – Optional custom workspace directory

Returns:

The initialized executor ready for step-by-step execution

Return type:

TaskExecutor

Example

```python # Start a task for step-by-step execution executor = start_task(

prompt=”Write a research report”, config_path=”config/team.yaml”

)

# Execute steps manually async for message in executor.start(prompt, stream=True):

print(f”Agent: {message.agent_name}”) print(f”Content: {message.content}”)

# You can inspect state, pause, or modify between steps if some_condition:

break

```
