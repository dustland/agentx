# AgentX Documentation

Welcome to **AgentX**, an open-source framework for building, observing, and orchestrating autonomous multi-agent systems.

## Quick Start

AgentX provides a powerful foundation for creating intelligent agent teams that can work together to solve complex problems.

### Key Features

- **Multi-Agent Orchestration**: Coordinate multiple AI agents seamlessly
- **Tool Integration**: Rich ecosystem of built-in and custom tools
- **Memory Management**: Persistent and contextual memory systems
- **Event-Driven Architecture**: Real-time communication and coordination
- **Observability**: Built-in monitoring and debugging capabilities

### Getting Started

To start using AgentX, install it via pip:

```bash
pip install agentx
```

Or clone the repository:

```bash
git clone https://github.com/dustland/agentx.git
cd agentx
pip install -e .
```
