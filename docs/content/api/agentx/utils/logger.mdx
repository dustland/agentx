Enable/disable streaming mode to control console output.


### get_logger

```python
def get_logger(name, level)
```

Get a configured logger instance.

Simple rule: - If streaming mode is active: AgentX loggers go to file only, others suppressed - If streaming mode is off: All loggers go to both console and file

**Arguments:**

- `name`: Logger name (usually __name__)
- `level`: Optional log level override

**Returns:**

Configured logger instance
