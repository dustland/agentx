# api

AgentX Server API

FastAPI-based REST API for task execution and memory management. Provides endpoints for creating and managing tasks, and accessing task memory.

```python
from agentx.server.api import ...
```

## Functions

### create_app

```python
def create_app(title, description, version, enable_cors)
```

Create and configure the FastAPI application.

**Arguments:**

- `title`: API title
- `description`: API description
- `version`: API version
- `enable_cors`: Whether to enable CORS middleware

**Returns:**

Configured FastAPI application


### add_routes

```python
def add_routes(app)
```

Add API routes to the FastAPI application


### run_server

```python
def run_server(host, port, reload, log_level)
```

Run the AgentX server with integrated observability.

**Arguments:**

- `host`: Host to bind to
- `port`: Port to bind to
- `reload`: Enable auto-reload for development
- `log_level`: Logging level
