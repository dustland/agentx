# monitor

AgentX Observability Monitor

Read-only observability system that monitors AgentX project data: - Reads workspace data from {project_path}/workspace/ - Reads configuration from {project_path}/config/ - Provides web interface for viewing project data - Does NOT create or modify any files

```python
from agentx.observability.monitor import ...
```
