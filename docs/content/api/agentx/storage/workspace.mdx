# workspace

Workspace storage - Business logic layer for workspace management.

Handles business concepts like execution plans, messages, artifacts, etc. Uses the filesystem abstraction layer underneath.

```python
from agentx.storage.workspace import ...
```

## Classes

### WorkspaceStorage

```python
class WorkspaceStorage
```

Workspace storage that handles business concepts.

Manages execution plans, messages, artifacts, and other workspace content using a filesystem abstraction underneath.
