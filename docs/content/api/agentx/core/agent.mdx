### Agent

```python
class Agent
```

Represents an autonomous agent that manages its own conversation flow.

Key Principles: - Each agent is autonomous and manages its own conversation flow - Agents communicate with other agents through public interfaces only - The brain is private to the agent - no external access - Tool execution is handled by orchestrator for security and control This combines: - AgentConfig (configuration data) - Brain (private LLM interaction) - Conversation management (delegates tool execution to orchestrator)
